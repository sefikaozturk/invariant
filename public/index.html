architecture
002

Three layers. One system.

I
Marketplace
The supply engine. Developers sell unused API credits P2P at 85-95% face value. Stripe escrow. Creates the supply that powers everything above.
AnthropicOpenAIGoogleMistral
II
Gateway
One endpoint, every API category. Picks best provider by price, latency, region. You call one endpoint. Routing is invisible.
AIMapsSMSEmailPaymentsStorage
III
Agents
Agent says "build me a restaurant finder." Gateway resolves geocoding, reviews, LLM. Provisions access, pays from balance. You funded $50. It cost $3.20.
discoverprovisionpaybuild
marketplace feeds supply → gateway consumes supply → agents drive demand → demand attracts sellers → cycle
